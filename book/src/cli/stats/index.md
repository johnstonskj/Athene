# Statistics

```bash
❯ hoot statistics --help
Compute statistics for the Ontology in an OWL document

Usage: hoot statistics [OPTIONS] <DOCUMENT>

Arguments:
  <DOCUMENT>  The documents to load

Options:
  -d <IMPORT_DEPTH>   If non-zero, process imported ontologies transitively to this depth [default: 0]
  -v, --verbose...    Increase logging verbosity by one level per occurance
      --quiet...      Decrease logging verbosity by one level per occurance
      --color <WHEN>  Controls when to use color [default: auto] [possible values: auto, always, never]
  -h, --help          Print help
```
